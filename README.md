
Aplicación web desarrollada con HTML, CSS y JavaScript que permite gestionar tareas de forma sencilla.

FUNCIONALIDADES
Añadir tareas
Marcar tareas como completadas
Eliminar tareas
Guardar tareas en el navegador (localStorage)

TECNOLOGIAS UTILIZADAS
 HTML
CSS
JavaScript
Manipulación del DOM
localStorage

USO DE LA APLICACIÓN
Escribir una tarea en el campo de texto
Pulsar el botón "Añadir"
Hacer click en la tarea para marcarla como completada
Pulsar "Borrar" para eliminarla

PERSISTENCIA DE DATOS
Las tareas se almacenan en el navegador mediante localStorage, por lo que no se pierden al recargar la página.

ESTRUCTURA DEL PROYECTO
* index.html = estructura de la aplicación
* style.css = diseño visual
* script.js = lógica de la aplicación
